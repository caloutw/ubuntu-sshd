# vscInit.sh 簡易使用說明

## 如何開啟

```bash
# 執行腳本
./vscInit.sh
```

## 工具介紹

### 快速設置
- **選項 0** - 開啟 SSL 安全憑證 (Code-server)
- **選項 5** - 安裝 tmux (背景多任務視窗)

### 其他工具
- **選項 31** - 安裝 Zip (壓縮/解壓縮工具)
- **選項 32** - 安裝 RClone (雲端儲存同步工具)

## 使用步驟

1. 執行腳本
2. 輸入對應數字選擇工具
3. 等待安裝完成
4. 按 `Ctrl + C` 退出
